A simple text-based (CLI) Blackjack game built with Python.
This project focuses on practicing game logic, functions, and clean program structure using core Python features.

It runs entirely in the terminal, no external libraries needed.

Features:
- Standard Blackjack rules (player vs dealer)
- Random card dealing from a shuffled deck
- Automatic dealer behavior (hits until 17)
- Win/Lose/Tie detection
- Clean and readable card display (not raw tuples)
- Replay option after each round

Built with Python 3 and standard random library (random).

NOTE: This is a learning project, not a full casino-accurate Blackjack simulator :)
